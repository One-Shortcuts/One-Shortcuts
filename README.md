# One Shortcut

Retail store daily operation tools for Apple Retail Thailand.

Production site:

- `https://i3udokai.github.io/One-Shortcut/`

## Files

- `index.html` - GitHub Pages entry point that redirects to `staff-tool.html`
- `staff-tool.html` - main launcher and embedded tool views
- `price-battle.html` - same-site Price Battle page
- `deploy_ShortcutTools.sh` - publishes deployable HTML files to the public `i3udokai/One-Shortcut` GitHub Pages repo

## Workflow

1. Edit the HTML files locally.
2. Validate with `bash -n deploy_ShortcutTools.sh`, `git diff --check`, and the inline-script parse command from `AGENTS.md`.
3. Run `./deploy_ShortcutTools.sh`.
4. Use the printed GitHub Pages URL in the iOS Shortcut.

## Notes

- `staff-tool.html` contains the launcher, Education Price, VAT Refund, Belkin claim, Highlight, Support, and Other views.
- Price Battle opens as `price-battle.html` on the same GitHub Pages site.
- The iOS Shortcut should open the GitHub Pages URL, not a Base64-encoded copy.
- The public Pages repo is HTML-only and is generated by the deploy script.
- Pages include `noindex` meta tags to discourage search indexing. This is not access control.
- The repo is intentionally framework-free and uses only static HTML/CSS/JS.
