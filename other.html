BANK TRANSFER

Bank Key มี 7 หลัก XXXYYYY

  • สามหลักแรก XXX คือ Bank Code (รหัสธนาคาร) ดูได้จากตารางด้านบน
  • สี่หลักหลัง YYYY คือ Branch Code (รหัสสาขา) ดูได้จากหน้าสมุดบัญชีของลูกค้า ถ้ารหัสสาขาของลูกค้ามีแค่ 2 หลัก ให้ใส่เลข 0 นำหน้าให้ครบ 4 หลัก เช่น รหัสสาขา 31 ให้ใส่เป็น 0031
  • ข้อมูลที่ใส่ต้องเป็นภาษาอังกฤษเท่านั้น
