BANK TRANSFER

Bank Key has 7 digits XXXYYYY Bank Key มี 7 หลัก XXXYYYY

  • First 3 digits XXX = Bank Code — see table above สามหลักแรก XXX คือ รหัสธนาคาร ดูได้จากตารางด้านบน
  • Last 4 digits YYYY = Branch Code — found in the customer's bank book. If fewer than 4 digits, pad with leading zeros (e.g., branch 31 → 0031) สี่หลักหลัง YYYY คือ รหัสสาขา ดูได้จากหน้าสมุดบัญชีของลูกค้า ถ้ารหัสสาขาของลูกค้ามีแค่ 2 หลัก ให้ใส่เลข 0 นำหน้าให้ครบ 4 หลัก เช่น รหัสสาขา 31 ให้ใส่เป็น 0031
  • All entries must be in English only ข้อมูลที่ใส่ต้องเป็นภาษาอังกฤษเท่านั้น
Example
ธนาคารไทยพาณิชย์ จำกัด (มหาชน)
SIAM COMMERCIAL BANK PUBLIC COMPANY LIMITED

Branch Code: 0714
Account No.: 714-299952-5
Savings Account
Note
  • If the branch code is unknown, ask the customer to call their bank's Call Center
  • In Isaac, some fields may not accept all data — that's fine, fill in as much as possible
  • The company name field does not accept spaces — type it without spaces
  • For foreign customers without a Thai bank account, partner with a Manager to process a cash refund
